API

As funcionalidades básicas da aplicação são interações com dados, i.e., visualização, criação, edição e remoção de informações armazenadas no banco de dados.

Tais ações são possibilitadas por uma API exposta ao usuário em um endpoint GraphQL no servidor web.

As Queries e Mutations que compõem a API são geradas pelo middleware PostGraphile, e correspondem a views e funções contidas no schema api do banco de dados.

Detalhes adicionais acerca de cada uma das Queries e Mutations, seus inputs, data types etc. podem ser visualizados na ferramenta GraphiQL.
