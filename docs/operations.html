Operações

A lista a seguir traz exemplos das operações que os USUÁRIOS podem executar nas entidades definidas anteriormente. A permissão para executá-las depende, principalmente, do NÍVEL DE ACESSO de cada USUÁRIO e dos atributos das entidades.

  • criar e editar ATIVOS
  • criar, editar e deletar TAGS
  • adicionar (ou remover) uma TAG a um ATIVO
  • estabelecer hierarquias de ATIVOS
  • criar e editar ESTOQUES
  • criar e editar INSUMOS
  • fazer uploads de arquivos (para TAREFAS, ATIVOS, ESTOQUES, PLANOS DE MANUTENÇÃO, FATURAMENTOS e PROJETOS)
  • criar e editar FATURAMENTOS
  • criar e editar NOTAS FISCAIS
  • adicionar (ou remover) os INSUMOS fornecidos a um FATURAMENTO
  • adicionar (ou remover) custos indiretos (e.g., depreciação) a um FATURAMENTO
  • criar e editar PLANOS DE MANUTENÇÃO
  • criar e editar ROTINAS DE MANUTENÇÃO
  • adicionar (ou remover) um ATIVO a uma ROTINA DE MANUTENÇÃO
  • criar e editar MONITORES
  • criar e editar medições referentes a um MONITOR
  • criar, editar e deletar um PROJETO
  • adicionar (ou remover) uma TAREFA a um PROJETO
  • criar e editar uma TAREFA
  • adicionar (ou remover) um ATIVO a uma TAREFA
  • propor a quantidade de um INSUMO a ser utilizado na execução de uma TAREFA
  • aprovar a quantidade de um INSUMO a ser utilizado na execução de uma TAREFA
  • indicar a efetiva quantidade de um INSUMO que foi utilizado em uma TAREFA
  • enviar uma TAREFA a uma EQUIPE
  • adicionar uma mensagem a uma TAREFA
  • alterar uma mensagem adicionada a uma TAREFA
  • alterar o status de uma TAREFA
  • adicionar (ou remover) uma TAREFA a uma lista pessoal de acompanhamento
  • criar e editar USUÁRIOS
  • criar e editar EQUIPES
  • adicionar (ou remover) um USUÁRIO a uma EQUIPE
  • editar informações do próprio perfil de USUÁRIO
  • trocar senha
